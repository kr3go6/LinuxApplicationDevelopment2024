## bin_search

bin search for guessing your numbers!  
Gussing your number using binary search: every time possible interval reduced in half depending on your answer.  

How to install:  

* autoreconf -vfis  
* ./configure  
* printf '\\n\\n\\n' | make  
* sudo make install-man  

# Usage: ./bin_search [-r] [-h]
        -h:     print this help  
        --help  
        -r:     roman mode
